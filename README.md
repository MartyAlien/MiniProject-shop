### 简单介绍

本项目是基于uniapp的一个商城小程序，后端接口采用java springboot进行搭建，实现了基本的购物车、收藏、登录、支付等功能

### 目录结构

```
|-- ec-server	后端接口
|   |-- docs
|   └-- src		后端代码逻辑
└-- uni-shop	uniapp小程序
    |-- components
    |-- js_sdk
    |-- pages	具体页面
    |-- request	封装的请求
    |-- static	静态资源文件
    |-- uni_modules
    |-- unpackage
    └-- utils

12 directories
```
